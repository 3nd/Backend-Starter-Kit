## Getting Started

### Clone ExpressTS-MongooseTS-Starter
```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/ExpressTS-MongooseTS-Starter.git
$ cd expressts-mongoosets-starter
```

### Install Dependencies
```bash
$ npm install
```

### Run the Application
```bash
$ mongod
$ npm start
```
