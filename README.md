## Getting Started

### Clone Express-Starter-Kit
```bash
$ git clone --depth 1 https://github.com/Shyam-Chen/Express-Starter-Kit.git <PROJECT_NAME>
$ cd <PROJECT_NAME>
```

### Install Dependencies
```bash
$ yarn install
```

### Run the Application
```bash
$ yarn start
```
